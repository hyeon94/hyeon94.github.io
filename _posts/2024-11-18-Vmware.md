**가상머신**  
기본적인 시스템 OS에서 알아봤으니 이제는 실제 환경 구축을 위해서는 가상머신 프로그램을 사용 해야된디.
실제로 있는 컴퓨터 시스템을 여러 명의 사용자가 동시에 사용할 수 있게 하기 위해서 사용한다.

 - **가상 머신(VM) 종류**  
    - **Virtual Box**
       - innotek에서 제장된 소프트웨어로 현재 오라클에서 상용 및 사유
       - 지원 : 리눅스, OS X, 솔라리스 윈도우
       - 32비트 가상화 소프트웨어
       - open source
    
    - **Parallels**
        - macOS 전용 가상화 프로그램
        - 개인 사용자에 특화

    - **VMware**
      - 많이 알려져 있는 상용 소프트웨어
      - 여러 가상머신을 다룰 수 있음
      - 지원되는 외부 디바이스 다양
      - 실제 PC에서 동작하지만 VMware에서는 동작하지 않는 경우도 있음
   

     - **QEMU**
       - Fabrice Bellard가 만든 소프트웨어
       - 32비트 이외의 기종을 위해 만들어진 스택 전체를 가상 머신 위에서 실행 가능
       - 동적 변환기 사용
       - 사용자 모드 에뮬레이션과 완전한 컴퓨터 시스템 모드 에뮬레이션을 지원
