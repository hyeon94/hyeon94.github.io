**반복문**

코드를 반복적으로 실행하고자 할때 사용하게 되는 제어문의 한 종류이다.

1) 종류

 - for문

        int num =0;
        for(int i = 0; i< 10; i++){   
        }
    
    i는 임의로 정한 변수이자 반복문에서 많이 사용되는 iterator이다.

    int i = 0;를 통해서 초기 값을 정한다.

    i< 10;로 i가 언제까지 반복할지 정해자는 조건을 정할 수 있다.

- while문

        (초기화);
        while(조건식){
        실행문;
        증감식;
        }

    while문은 조건이 충족되는 경우에 계속해서 실행문을 반복 실행시킨다. 초기화와 증감식의 위치는 for문과 다르지만 상호 대체가 가능하다.

- do-while문

        do {
           실행문 
        } while (조건식);

    do-while문은 조건식에 의해 실행문을 반복 실행된다.While문은 조건식을 시작할 떄 검사해 브록 내부의 실행 여부를 결정하지만 do-while문은 실행의 순서가 다르다. 우선 실행문을 모두 실행하고 조건식을 평가하는 순서이다.결과가 참이면 실행문과 조건식을 반복 실행하고, 거짓일 경우 종료된다.
