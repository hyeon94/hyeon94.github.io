**함수(Function)**

하나의 특벽한 목적의 작업을 수행하도록 설계된 독립적인 블록을 의미한다.
이러한 함수는 필요할 때마다 호출하여 해당 작업을 수행할 수 있다.

![](/assets/img/sub/사진(17).png)


    리턴타입 함수이름(매개 변수){ //함수의 선언부
      //함수의 구현부
    }

**메소드(Method)**

객체의 기능을 구현하기 위해서 클래스 내부에 구현되는 함수이다.
맴버 함수하라고도 하며, 객체의 기능이 구현된다.

    리턴타입 함수명(자료형 a, 자료형 b){
        return 결과값;
    }
