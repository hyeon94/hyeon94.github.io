**반복문**
반복문이란? 어떤 조건에 만족할 떄까지 같은 처리를 반복하여 실행하도록 하는 제어문이다.

 + **종류**
    + **while문**
     조건에 떄라 반복할 떄 사용할 경우, true일 경우 반복 false일 경우 종료

     ```java
     while(조건문){
    
        실행문;
     }
     ```

     + **do-while문**
     while문과 유사하나 조건을 나중에 검사한다.

     ```java
     do {
        실행문;

        while(조건식);
        // 해당 조건식이 true일 경우 반복, false일 경우일 탈출
     } 
     ```
