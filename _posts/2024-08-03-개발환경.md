**JDK 설치**

- JDK란?  
  자바로 소프트웨어 개발할 수 있도록 도와주는 패키이다.

- 종류 
  - Java SE
    표준 자바 플랫폼으로 표준적인 컴퓨팅 한경을 지원하기 위한 자바 가상 머신 규격 및 API 집함을 포함한다.

  - Java EE  
    구체적인 목적에 따라 Java SE를 기반으로 API를 추가하거나 자바 가상머신 규격 및 API 일부를 택하여 정의한다.
    Java SE에 웹 어플리케이션 서버에 동작하는 기능을 추가한 플랫폼이 스펙에 따라 제품을 구현한 것을 웹 어플리케이션이라고 한다.

  - Java ME  
    제한된 자원을 가진 휴대전화,PDA 등에서 Java 프로그래밍 언어를 지원하기 위해 만든 플랫폼 중 하나이다.
