**조건문**
조건문이란? 주어진 조건을 비교 판단하여 그 조건을 만족할 경우 명령을 실행하고, 만족하지 않을 경우 다음 단계의 명령을 수행하도록 하는 제어문이다.

  + **종류**
    + **단순 if문**
    ![](assets/img/photo/java(3).png)

    ```java
      if(조건식){
        실행문;
        실행문;

      }    
    ```

    + **if-else문**
        ![](assets/img/photo/java(4).png)

        ```java
         if(조건문){
            true 일경우 실행;        
            }else{
             false 일 경우 실행;
            }
        ```

    + **다중 if-else문**

        ![](assets/img/photo/java(5).png)
 
        ```java
        if(조건문1){
            조건문1이 true일 경우 실행;
        
        }else if(조건문2){
            조건문2가 true일 경우;
        
        }else if(조건문3){
            조건문3가 true일 경우;

        }

        ```
    +  **Switch-case문**
      ```java
      switch(변수){
        case 값 1:
      break;
         case 값 2:
      break;
      
      }
      ```
