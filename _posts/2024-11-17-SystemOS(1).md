**운영체제 역할**
 - **자원관리**  
    - **프로세스**  
      운영체제는 시스템 상에서 실행되는 모든 프로세스를 관리한다. 여기서 프로세스란 실행 중인 
      프로그램의 인스턴스이다. 운영체제는 프로세스 생성 스케줄링 종료 등 관리하여 다수의 프로
      세스가 효율적으로 실행될 수 있도록 도와준다.

    - **메모리**  
      RAM(랜덤 접근 메모리)와 같은 주 메모리의 할당 및 회수를 관리한다.이는 메모리 공간을 효율적
      으로 사용하고 프로세스 간 메모리 충동을 방지
    
    - **디스크 및 스토리지**  
      파일 시스템을 통해 하드 드라이브와 같은 스토리지 장치에 데이터를 저장하고 접근하는 방법 제공한다.
      이를 통해 데이터의 안전한 저장과 효율적인 접근이 가능하게 한다.

    - **장치 관리**  
       컴퓨터에 연결된 모든 입출력 장치의 관리를 담당한다. 이는 장치 드라이버를 통해 이루지며, 사용자와 응용
       프로그램이 하드웨어 장치를 쉽게 사용할 수 있도록 해준다.

    - **파일시스템 관리**  
       파일과 디렉토리를 조직하고 관리하는 파일 시스템을 제공하낟. 이는 사용자와 응용 프로그램이 파일을 생성 저장 검색 수정할 수 있도록 도와준다.

    - **보안 및 접근제어**  
       시스템의 보안을 유지하기 위해 사용자 인증, 접근 권한 설정, 암호화 등의 기능을 제공한다. 이는 무단 접근으로부터 시스템과 데이터를 보호한다.   

    - **네트워킹**  
      네트워크 통신을 가능하게 하는 기능을 포함한다. 이는 컴퓨터가 네트워크에 연결되어 데이터를 송수신할 수 있도록 하는 기능을 의미한다.
